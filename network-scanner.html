Network Scanner

A cybersecurity tool developed in Python that scans networks to identify connected devices, open ports, and potential vulnerabilities, helping administrators secure their networks.

Key Features

  • Network device discovery and mapping
  • Port scanning with service detection
  • Vulnerability assessment
  • Detailed reporting capability
  • Custom scan profiles

Technical Details

Technologies Used

  • Python 3.8+
  • Scapy library
  • Nmap integration

Security Features

  • MAC address spoofing
  • Stealth scanning techniques
  • Rate limiting to avoid detection

Challenges & Solutions

Network Permissions

Implemented privilege escalation checks and sudo handling to ensure proper permissions for low-level network operations.

Performance Optimization

Developed a multithreaded scanning approach with configurable thread pools to balance speed and resource usage.
